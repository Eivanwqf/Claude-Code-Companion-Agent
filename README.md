# Claude-Code-Companion-Agent
实际上如果想搭建一个能陪伴你聊天，能承接你的情绪的Agent，这些一样重要：时间感知能力、记忆能力、及时纠错的能力、可以换个视角为你提供思维的能力。至于本文说的Claude Code自带的自动记忆，它在平时写代码、教学知识的时候表现不赖。但是，对于每天日常的情绪和存储在记忆路径中的具体事件，他是很少能自动获取到的。其最根本的差异就是，自带的只会通过阅读description这一行字段，来判断这个记忆是否上下文相关。可是实际上，现实生活中发生的事情，很多细节是无法通过description一行来表达清楚的。所以，我不希望这种事情发生，我希望他能记住我所有发生过的事情——就像一个老朋友一样。
